Editing the Template Module

Edit the content of this template page to make a new module. You may edit the positioning of this sidebar_template aside element in the style css file. You may also edit the icon image of this aside element by chaning the image source path. Make sure to insert module links for the next and back buttons so they link to the proper modules. There are a set of navigation buttons at both the top and bottom of this page.
